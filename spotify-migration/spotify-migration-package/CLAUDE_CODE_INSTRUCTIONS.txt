STEP 1: Download the Files
---------------------------
Download the spotify-migration.tar.gz file to your local machine.

STEP 2: Extract the Archive
---------------------------
# On macOS/Linux:
tar -xzf spotify-migration.tar.gz
cd spotify-migration-package

# On Windows (using Git Bash or WSL):
tar -xzf spotify-migration.tar.gz
cd spotify-migration-package

# Or use a GUI tool like 7-Zip on Windows

STEP 3: Open in Claude Code
---------------------------
# Navigate to the directory in your terminal
cd /path/to/spotify-migration-package

# Start Claude Code
claude-code

# Or if you prefer VS Code with Claude
code .

STEP 4: Tell Claude Code What to Do
---------------------------
Once in Claude Code, simply paste this prompt:

"""
I have a MongoDB database for a Spotify listening history tracker that's 
running out of storage. I need to migrate to PostgreSQL to save ~80% space.

I have migration files in this directory:
- schema.sql (optimized PostgreSQL schema)
- migrate.js (migration script)
- verify.js (verification script)
- README.md (full documentation)

Please help me:
1. Review the migration files and explain what they do
2. Set up my PostgreSQL database (I want to use [Supabase/Neon/etc.])
3. Configure my .env file with the right connection strings
4. Run the migration safely with a dry run first
5. Verify the migration was successful

My MongoDB connection string is: [YOUR_MONGODB_URI]
"""
